📌 Page 1 — 노인 이해·인구 특성
① 노화(aging)의 개념
  • 노화(aging) — 나이가 많아지는 가령 현상 (질병이나 사고 없이 시간의 흐름에 따라 생체 구조가 쇠퇴하는 현상)
  • 넓은 의미: 출생 후 시간의 경과에 따라 정상적으로 일어나는 모든 변화의 총체, 출생과 동시에 시작
  • 노화의 부정적 측면: 신체적·심리적·정신적·사회적 기능 저하
  • 노화의 긍정적 측면: 경험과 지혜의 축적, 숙련도 향상, 활동 능력 확대
  • 노화의 고유성: 피할 수 없지만 반드시 연령과 일치하지 않는 개인차가 있음

④ 노인 인구의 특성
주요 특성
  • 평균 수명 증가 — 의료기술 발달, 위생 개선
  • 여성 노인: 평균 수명 남성보다 길어 경제적 어려움, 독거 기간 길어짐
  • 만성 질환 유병률 — 고혈압(56.7%)>관절염>당뇨병>요통·좌골통>고지혈증>골다공증 순
  • 심장 질환으로 인한 사망률 감소, 암으로 인한 사망률 증가
  • 노인 인구 의료비 증가율 > 전체 인구 의료비 증가율
노년 부양비 & 노령화 지수
  • 노년 부양비 = 65세 이상 인구 ÷ 15~64세 인구 × 100
  • 노령화 지수 = 65세 이상 인구 ÷ 0~14세 인구 × 100
  • 노령화 지수 증가 → 노년 부양비 증가
  • 2015년: 생산 가능 인구 5.7명이 고령자 1명 부양
  • 2030년 전망: 생산 가능 인구 2.6명이 고령자 1명 부양
소외와 외로움 문제
  • 은퇴 후 중심 역할 상실, 사회적 관계망 축소 → 심리적 소외감
  • 자녀와 떨어져 살고 배우자 사별 후 독거 → 역할 상실·소외감 심화
📌 Page 2 — 노화에 따른 신체 변화 I (세포·근골격·순환·호흡·소화)
① 세포 변화
  • 세포 수 감소 → 기능하는 세포 수 감소, 세포·뼈의 질량 감소
  • 세포 외액은 일정하게 유지, 세포 내액 감소 → 총 체액량 감소 → 탈수 위험
② 근골격계 변화
  • 척추후만증, 관절 비대, 근육 처짐, 신장 감소
  • 근육량·근력·기능 감소 — 비활동적인 노인에게 주로 → 운동 필요
  • 건(腱)이 위축·딱딱해져 건반사 감소·소실, 그러나 무릎 반사는 유지
  • 골 흡수량 > 골 생성량 → 칼슘 밀도 낮아짐 → 골다공증·골절 발생 쉬움
  • 장골이 짧아지지는 않지만 얇아진 디스크 + 짧아진 척추뼈 → 척추 길이 짧아지고 키 줄어듦
  • 관절 연골 표면 퇴화, 끝이 뾰족하거나 돌출 → 관절 움직임·동작에 제약
노인 권장 근력 강화 운동: 수영, 팔굽혀 펴기, 윗몸 일으키기, 가벼운 아령·모래주머니·고무 밴드
③ 순환기계 변화
  • 심장 크기: 노화보다 심장 질환으로 비대, 두드러진 활동 부족은 심장 위축
  • 방실판(atrioventricular valves) — 경화·섬유화로 두꺼워지고 강직
  • 판막 불완전 폐쇄 → 수축기·이완기의 잡음
  • 심박출량·일회 박출량 감소 + 심내막·좌심실 섬유화 → 체위성 저혈압 유발
  • 심장 박동 수: 스트레스 시 증가하나 청년보다 증가 정도 적음, 정상으로 돌아오는 속도 느림
  • 혈관 내막의 섬유화·칼슘·지방 축적·세포 증식 → 죽상동맥 경화증으로 발전
  • 압력 수용기 기능 장애 + 말초 저항 증가 → 수축기 혈압 증가
  • 혈관 탄력 감소 + 피하 지방 감소 → 머리·목·사지 혈관 돌출
④ 호흡기계 변화
  • 코 끝이 아래로 처지고 비중격 만곡증 → 코골이·수면 무호흡증
  • 늑연골 석회화 → 기도·흉곽 강직, 가슴 전후경(AP diameter) 증가, 척추후만증, 호흡 근력 약화
  • 폐의 섬모 수 감소 + 기관지 점액샘 비대 → 가래·분비물 배출 능력 약화
  • 폐포 탄력성 감소, 폐 강직성 심해짐, 기저막 팽창 불충분 → 이물질 배출 능력 감소
  • 폐의 비효과적 호기 → 잔기량 증가, 폐활량 감소 → 부동 시 호흡기 감염 고위험군
  • 지구력 트레이닝은 노인의 폐활량 증가 가능
⑤ 소화기계 변화
  • 치아: 치주 질환 빈도 증가, 치아 상실 시 의치 사용 → 불편감 / 식욕·소화 기능 감퇴
  • 혀: 위축·만성 자극으로 미각 서서히 약화, 특히 단맛 기능이 다른 맛보다 많이 둔해짐
  • 노인은 성인에 비해 약 1/3의 침 생성 → 구내 건조증 흔히 발생
  • 식도 운동성 저하, 약간 늘어남, 식도 하부 조임근 이완 → 흡인 위험
  • 소장·대장: 전반적 위축, 소장벽 영양 흡수 세포 감소, 대장 점액분비·직장벽 탄력 감소
  • 노화로 인한 대장 운동력 변화는 발생하지 않지만 섬유소·수분 부족, 운동 부족 → 변비 자주 발생
  • 간의 크기 감소, 손상된 간세포 재생력 저하, 비효율적 콜레스테롤 대사 → 담석 발생 빈도 증가, 약물 중독 가능성 높아짐
📌 Page 3 — 노화에 따른 신체 변화 II (비뇨·생식·내분비·신경·감각·피부)
① 비뇨기계 변화
  • 네프론 수 감소 → 신장 조직 손실 → 사구체 여과율 감소
  • 세뇨관 기능 감소 → 물질 교환 능력, 물·나트륨 보존 능력 저하, 당 재흡수 감소 → 단백뇨
  • 방광 근육 약화·기능 감소 → 빈뇨·긴박뇨·야뇨증
  • 다산한 여성: 골반 근육 약화 → 긴장성 요실금
② 생식기계 변화
남성
  • 정낭의 세포액 유지 능력 감소, 정세관의 정자 수 감소
  • 오르가즘·사정 강도 줄어들지만 발기·사정 능력은 유지
  • 보편적으로 전립선 비대 (대부분 양성, 악성 발전 위험 → 정기 검사)
  • 65세 이상의 3/4은 전립샘증 → 빈뇨 원인
여성
  • 호르몬 변화 → 외음부 위축, 피하지방·음모 감소, 음순 위축
  • 질 내부: 건조하고 알칼리화 → 균 무리·분비물 감소
  • 자궁 크기 감소, 자궁내막 위축 (호르몬 반응 지속)
  • 에스트로겐 감소 → 골반 저부 근육 약화 → 복압 증가 시 불수의적 소변 배출
③ 내분비계·신경계 변화
내분비계
  • 갑상샘 기능 저하 → 기초대사량 감소
  • 생식선 분비 감소 → 테스토스테론·에스트로겐·프로게스테론 점진적 감소
  • 인슐린에 대한 세포 민감성 감소 → 인슐린 분비 지연·비효과적 방출 → 당 대사 능력 감소
  • 노인에게서 혈당이 높게 측정되는 경우 흔함 (당뇨병이 아닌 경우)
신경계
  • 뇌세포: 오랜 시간에 걸쳐 감소, 특히 55세 이후 뇌의 크기·무게 감소
  • 신경전도 속도 느려짐 → 반사 느려짐, 인지·반응 지연, 단기기억능력 많이 감소
  • 균형 반응 느려짐 → 낙상 원인
  • 수면 양상 변화: 전체 수면 시간 짧아짐
  • REM 수면: 일정하게 유지 / NREM 수면: 짧아짐
④ 감각 기계 변화

⑤ 피부계 변화
  • 피부 탄력성 감소, 건조, 푸석푸석해짐, 피하 지방 감소 → 주름·처짐
  • 노인성 반점(검버섯) — 멜라닌 세포가 뭉쳐 있는 것
  • 피부 면역 기능 감소 → 피부 감염에 쉽게 노출
  • 두피·치골·겨드랑이 털: 얇아지고 회색, 그러나 코·귀의 털은 더 두꺼워짐
  • 손톱: 성장 느려지고 깨지거나 부서지기 쉬움, 세로선 발달, 손톱 반달 크기 감소
  • 땀샘 수·기능 감소 → 발한 약간 줄어듦
📌 Page 4 — 심리·사회적 변화
① 에릭슨(Erikson)의 노년기 발달 과업
통합성 (integrity) 성취
  • 자신의 인생 경험을 긍정적으로 보며 과거와 현재의 인생을 수용
  • 성숙함 보임, 과거의 생활 유형 수용, 평온
  • 자아 실현 계속하면서 죽음을 수용할 수 있게 됨
통합성 미성취
  • 과거에 대한 후회와 원망
  • 비현실적이고 도달하기 어려운 목표 설정
  • 죽음에 대한 공포와 절망
② 노년기 성격 변화의 특징 (7가지)
  • 조심성 증가: 정확성 중시, 결정에 자신감 결여
  • 경직성 증가: 새로운 변화 싫어하고 융통성 없어짐
  • 우울 성향 증가: 신체 질병, 배우자 사망, 경제 사정 악화, 사회 고립
  • 생애에 대한 회고 경향: 해결하지 못한 것 찾아 새로운 해결 시도
  • 친근한 사물에 대한 애착심 증가: 과거 회상, 마음의 안정
  • 성역할 지각 변화: 남성 → 친밀감·의존성 증가 / 여성 → 공격성·자기주장 증가
  • 의존성 증가
③ 노인의 심리·사회적 변화와 적응 방법
생활 사건심리·사회적 변화적응 방법
생활형태·가족 유대 변화
(자녀 성장, 핵가족화)		 심리적 상실감, 역할 변화 (부모→조부모) 조부모 역할로 만족감 / 정기적 부모-자녀 접촉
은퇴(퇴직) 역할 상실, 수입 감소, 사회 생활 축소 퇴직을 자유·기회·성장의 시기로 이해 / 여가 선용 / 감소된 수입 현명하게 관리
배우자의 죽음, 의미 있는 사람의 죽음 애도 및 상실 과정 (심리적), 독신 생활 (사회적) 배우자 의존도 높았던 남성에게 더 적응 어려움 / 친구·지역 모임 활용
건강 상태 변화 (기능 쇠약) → 질병과 무능력 자신의 신체상·자아 개념 변화, 죽음에 대한 두려움 신체 기능 감퇴 이해, 최대한 독립성·위엄을 지니며 목표 세움
④ 노인의 시각 변화에 따른 간호
시각 장애가 있는 경우
  • 대화 시작 시 자신의 이름 소개
  • 주변 환경 설명 (낯선 환경에 불안하지 않도록)
  • '저기·거기·이것' 등 지시 대명사 사용 금지
  • 구체적인 사물의 이름 사용
  • 점자·문자 해독기 익숙한 경우 활용
청각 장애가 있는 경우
  • 소음 제한 → 조용한 환경 조성
  • 밝은 곳에서 같은 높이, 마주보고 앉기
  • 대화 시 고음 피하고 천천히 분명하게 발음
  • 필요 시 글로 써서 의사 표현
  • 대화 중 얼굴에 손 대거나 껌 씹는 행동 삼가기
  • 짧은 문장으로, 다음 문장 전에 약간 간격 두기
⑤ 인지 기능 장애가 있는 경우
  • 시계·달력 같은 시각적 기구 활용
  • 이름과 존칭 사용 → 주체성·정체성 강화
  • 서두르지 않고 안정된 환경 유지
  • 간단하고 익숙한 단어를 사용하고 정확하게 천천히 말함
  • 주변의 불필요한 자극 제거
  • 의사소통 중 배설·배고픔 등 기본욕구 불충족으로 인한 불안 여부 확인·해결
  • 노인이 적절한 단어를 떠올리지 못할 경우 관련 단서 제공
  • 노인이 한 말의 내용이 사실이 아니더라도 수용

문항내용정답
01 노인 인구의 특성에 대한 설명으로 옳지 않은 것은?
① 다양한 형태의 여가 활동 ② 평균 수명 증가 ③ 노령화 지수·부양비 증가 ④ 서구보다 빠른 고령화 ⑤ 만성 질환 사망원인		 ① (노인 여가 활동은 단순·오락 위주)
02 근골격계 변화 (가), 소화기계 변화 (나) 관련 옳은 것은?
ㄱ. 추간판 얇아져 키 줄어든다 ㄴ. 구내 건조증 발생 ㄷ. 손톱이 깨지거나 세로선 발달 (피부계)		 ④ ㄱ, ㄴ (ㄷ은 피부계)

05 UN 고령화 사회 분류 ① 고령화사회(7~14%) ② 고령사회(14~20%) ③ 초고령사회(20% 이상)

② 빈출 암기 포인트 🔥
✦ 핵심 암기
  • 한국 고령화 사회 진입: 2000년 / 고령사회: 2017년 / 초고령사회: 2026년
  • 한국 고령화 소요 연수: 7→14% = 17년 / 14→20% = 9년 (세계 최빠른 수준)
  • 노인 만성 질환 1위: 고혈압(56.7%)
  • 노화: 골 흡수량 > 골 생성량 → 골다공증 (반대로 기억하지 않도록!)
  • 건반사: 감소하거나 소실 / 무릎 반사: 유지
  • 청각 변화: 2,000Hz 이상 고주파가 가장 먼저 소실
  • 미각 변화: 혀 뒤쪽 쓴맛·신맛 기능 증가 / 앞쪽 단맛 기능 감소
  • 소화기계: 대장 운동력 변화 없음 (변비는 섬유소·수분·운동 부족 때문)
  • 노인 타액: 성인의 약 1/3 → 구내 건조증
  • 수면 변화: REM 수면 유지 / NREM 수면 감소
  • 노년기 성격: 조심성·경직성·우울성향·의존성 증가
  • 에릭슨 노년기 발달 과업 = 통합성(integrity)
③ 자주 틀리는 선지 ⚠
틀리기 쉬운 것들
  • 노인 여가 활동은 다양함 → ✗ (단순하고 오락 위주)
  • 골 흡수량 < 골 생성량이어서 골다공증 → ✗ (골 흡수량 > 골 생성량)
  • 노화로 대장 운동력 변화 발생 → ✗ (대장 운동력 변화는 발생하지 않음)
  • REM 수면 시간이 짧아짐 → ✗ (REM은 유지, NREM이 짧아짐)
  • 노년기에는 경제적 독립 달성 → ✗ (의존도 증가)
  • 청각 저주파부터 소실 → ✗ (2,000Hz 이상 고주파부터)
  • 노인 단맛 기능이 예민해짐 → ✗ (단맛 기능 감소, 쓴맛·신맛 증가)
  • 손톱 세로선 발달은 근골격계 변화 → ✗ (피부계 변화)
