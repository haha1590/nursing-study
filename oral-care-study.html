📌 Page 1 — 구강의 구조
① 턱뼈
위턱뼈 (상악골; maxilla)
  • 위턱 부분에 있는 한 쌍의 뼈
  • 경구개(입천장 중 단단한 전방부)와 상악동(부비동의 한 종류)이 존재
  • 움직이지 않는 고정된 뼈
아래턱뼈 (하악골; mandibula)
  • 아래턱 부분에 위치
  • 두개골 중 유일하게 움직일 수 있는 뼈
  • U자 또는 L자 형태
② 치아 & 측두 하악관절(TMJ)
치아
  • 소화계통의 시작 담당
  • 상악골과 하악골에 좌우 대칭의 활 모양(치열궁)으로 배열
치아의 기능
  • 저작 — 전치(앞니): 음식 물어 자르고 찢기 / 구치(어금니): 잘게 씹기
  • 발음 — 혀, 입술과 함께 소리 냄
  • 심미 — 치아 유무·배열이 얼굴 모양에 영향
  • 연하 작용 — 음식물 삼키는 데 도움
측두 하악관절 (TMJ)
  • 측두골과 하악골로 구성
  • 하악골이 상하·좌우·전후로 움직일 수 있게 해줌
침샘 (타액선; salivary gland)
  • 대타액선: 이하선(귀밑샘) > 악하선(턱밑샘) > 설하선(혀밑샘)
  • 소타액선: 크기가 아주 작고 수가 많음
암기! 타액선 크기 순서 = 이하선 > 악하선 > 설하선
③ 치아의 구조와 조직
치아의 구조 (3부위)
  • 치관(crown) — 치아의 머리 부분, 육안으로 보이는 부분
  • 치근(root) — 치아의 뿌리 부분, 턱뼈에 박혀 보이지 않음
  • 치경(cervix) — 치관과 치근이 만나는 부분 (치경선)
치아의 조직 (4가지)
  • 사기질(법랑질; enamel) — 인체에서 가장 단단한 부분, 치관 가장 바깥쪽 / 불소가 가장 잘 침착하는 부위
  • 상아질(dentine) — 치관·치근 모두 존재, 치아 대부분 구성 / 경도 약해 우식증 시작되면 빠르게 진행
  • 백악질(cementum) — 치근 가장 바깥쪽, 치주인대와 함께 치아를 턱뼈에 결합
  • 치수(pulp) — 치아 가장 가운데(치수강), 신경과 혈관으로 구성
④ 치주 조직 (periodontal tissue)
경조직
  • 턱뼈(치조골) — 치아가 자리잡은 상악골·하악골, 치아 지지·보호
연조직
  • 치은(잇몸; gingiva) — 치조골 덮어 보호하는 섬유성 상피조직, 건강 상태: 밝은 분홍색
  • 치주인대(치근막) — 백악질과 함께 치아를 턱뼈에 결합, 저작 시 교합압 감소
  • 혀(tongue) — 음식과 침 섞기, 미뢰(미각세포) 존재, 발음 도움
⑤ 치아의 맹출
유치
  • 치배: 태생 7~8주부터 형성
  • 생후 6개월 정도에 맹출 시작
  • 2~3세 정도에 모두 맹출 (총 20개)
  • 교환 순서: 하악 유중절치부터 시작 → 제2유구치와 제2소구치 교환으로 끝
영구치
  • 치배: 태생 20주에 형성
  • 6세 정도에 맹출 시작
  • 12~13세에 제3대구치 제외하고 모두 맹출 (총 32개)
  • 맹출 순서: 상악 6-1-2-4-5-3-7-8 / 하악 6-1-2-3-4-5-7-8
  • 제1대구치(6세구치): 맹출이 너무 빨라 유치와 혼동되어 충치 잘 생김
  • 제3대구치: 사랑니(지치), 사춘기 이후 맹출
📋 치열 구분
  • 유치열: 구강 내 유치만 배열
  • 혼합치열: 유치와 영구치가 섞여 있는 상태 (6세~11세)
  • 영구치열: 구강 내 영구치만 배열 (12세 이후)
📌 Page 2 — 치아 이름·치식·해부학적 용어
① 치아의 이름
영구치 이름 (정중선 기준, 가까운 쪽부터)
1. 중절치 2. 측절치 3. 견치 4. 제1소구치 5. 제2소구치 6. 제1대구치 (6세구치) 7. 제2대구치 (12세구치) 8. 제3대구치 (사랑니)
유치 이름 (정중선 기준, 가까운 쪽부터)
유중절치 유측절치 유견치 제1유구치 제2유구치
치아 이를 때: 상하악 → 좌우측 → 치아 이름 순서 (예: 상악 우측 제1소구치)
② 치식 (Palmer Notation System)
팔머 표기법 (Palmer)
  • 정중선 기준으로 영구치: 1번~8번
  • 유치: A~E
  • 상하악 4분면으로 구분하여 표기
예) 상악 우측 제2소구치 = 5┘ / 하악 좌측 측절치 = └2
하악 우측 유견치 = C┘ / 상악 좌측 제1유구치 = └D
FDI 표기법 (국제치과연맹)
  • 영구치: 상악 우측 #11~#18 / 상악 좌측 #21~#28
  • 하악 좌측 #31~#38 / 하악 우측 #41~#48
  • 유치: 상악 우측 #51~#55 / 상악 좌측 #61~#65
  • 하악 좌측 #71~#75 / 하악 우측 #81~#85
예) 상악 우측 제2대구치 = #17 / 하악 좌측 견치 = #33
③ 치아의 해부학적 용어
용어설명
순면/협면 (labial/buccal)치아의 면 중 입술(순)·뺨(협) 쪽에 위치한 면
설면 (lingual)치아의 면 중 혀(설) 쪽에 위치한 면
인접면 (proximal)옆 치아와 닿아 있는 면 (근심면 + 원심면)
근심면 (mesial)인접면 중 정중선에 가까운 쪽 면
원심면 (distal)인접면 중 정중선에 먼 쪽 면
절단연 (incisal)전치부 치관의 가장 끝 부분, 날카롭게 되어 있는 모서리 형태의 연(면)
교합면 (occlusal)구치부 치관의 가장 끝 부분, 음식 씹고 부수도록 울퉁불퉁한 면 (전치부 절단면에 해당)
교두 (cusp)구치부 교합면에 둥글게 튀어나온 부위
와 (fossa)치아 면에서 움푹 들어간 부위
소와 (pit)와 내에서 가장 깊은 부위
구/열구 (groove/fissure)치아 면에서 선 모양으로 들어간 부위
📌 Page 3 — 구강 질환·예방
① 치아 우식증 (충치; dental caries)
원인 (4가지)
  • 뮤탄스균(mutans) — 탄수화물 분해 → 산(acid) 생성 → 치아 우식 유발
  • 치면 세균막(치태) — 치아에 부착된 연성 물질 (회색·노란색), 타액·상피세포·음식물 잔사 포함
  • 탄수화물 — 아밀라제에 의해 분해되며 우식 유발 세균의 영양분
  • 타액의 점성 증가 — 구강 자정 작용 저하 → 우식 증가
충치 예방법
  • 상수도 불소화법 — 수돗물에 1ppm 불소 → 우식 60% 예방
  • 전문가 불소 도포 — 불화나트륨 고농도 치아 표면 도포
  • 치아 홈 메우기(실란트) — 교합면 열구·소와를 레진으로 메움, 광중합 레진 사용
  • 치면 세균막 관리 — 칫솔·치실·치간 칫솔 사용
  • 식이 조절 — 설탕 제한
  • 정기 치과 검진
  • 저탄수화물 식이
반상치: 에나멜질 형성 부전 → 음료수 불소 1ppm 이상 시 발생 (치아표면 유백색 줄무늬)
📋 칫솔질 방법·시기
  • 칫솔 선택: 중간 정도 탄력, 머리는 작고 둥글게, 2~3달에 한 번 교체
  • 칫솔을 치아와 잇몸이 만나는 곳에 위치
  • 상악: 위에서 아래로, 하악: 아래서 위로 (회전법)
  • 10초 정도 반복 후 이동
  • 혀와 구강 점막 부위도 닦기
  • 치실·치간 칫솔로 치아 사이 닦기
  • 칫솔질 시기: 음식 섭취 후 즉시, 하루 최소 3번 이상
  • 24시간 지나면 치면 세균막 침착 다시 일어남
② 치주 질환 (periodontal disease)
치은염 (gingivitis)
  • 치주 질환의 초기
  • 치은(잇몸) 조직의 염증
  • 증상: 발적, 부종, 출혈, 통증
  • 원인: 세균성 염증, 치석, 당뇨 등
치주염 (periodontitis)
  • 치은염이 만성적으로 진행
  • 치주낭 형성, 치은 퇴축(치은 흡수)
  • 치조골 퇴축, 치아 주위조직 소실
  • 심해지면 치아가 흔들려 발치 가능
③ 부정 교합 (malocclusion)

상하악 치아의 배열이나 맞물림이 비정상적인 상태 (숫자는 심한 정도와 관계없음)

급수특징속칭
1급 (class Ⅰ) 상하악 제1대구치 위치 관계는 정상 / 나머지 치아가 배열 이상·돌출·회전 —
2급 (class Ⅱ) 상악이 정상 기준보다 전방으로 돌출
· 1류(div 1): 상악 전치부 전방 돌출
· 2류(div 2): 상악 전치부가 설측으로 기울어짐		 뻐드렁니 / 옥니
3급 (class Ⅲ) 하악이 정상 기준보다 전방으로 돌출 / 2·3급은 턱뼈 성장 불균형 → 초등학교 전 교정 시작 주걱턱
부정교합의 기준: 상악 제1대구치와 하악 제1대구치의 위치 관계
원인: 유전, 습관(손가락 빨기), 유치의 만기 잔존·조기 탈락, 빠진 영구치 방치
④ 구강 건조증 & 치면 세마
구강 건조증
  • 주로 노년기에 발생
  • 타액 분비 저하로 불편함
  • 증상: 음식 삼키기 힘듦, 말하기 힘듦, 구취
  • 지속 시 → 치아우식증, 치은염, 치주염, 혀 갈라짐 악화
치면 세마 (스케일링)
  • 치태가 제거되지 않아 단단하게 석회화된 것 = 치석
  • 치석은 칫솔질로 제거 안 됨 → 치과에서 큐렛·시클·초음파 스케일러로 제거
  • 만 19세 이상 건강보험 1회/1년 적용 (1월 1일 ~ 12월 31일)
스케일링 구분: ① 치석 제거 ② 치면세마 (치태·음식물잔사·외인성색소·치석 제거) ③ 치근 활택
⑤ 3차 예방 구분
1차 예방 (건강증진)
  • 칫솔질
  • 불소 이용법
  • 식이 조절
  • 치아 홈 메우기
  • 치면 열구 전색
  • 구강보건 교육
2차 예방 (조기발견·치료)
  • 구강 검진
  • 치은염 치료
  • 치석 제거
  • 충전 치료
    (법랑질·상아질 우식)
3차 예방 (기능 회복)
  • 치수 치료(신경 치료)
  • 치주염 치료
  • 치아 발거
  • 의치
  • 보철
📌 Page 4 — 치과 기구·장비·진료 보조
① 기본 기구 (3가지)
치경 (mirror)

눈으로 보이지 않는 부위를 관찰하거나 빛을 반사하여 구강을 직접 관찰

탐침 (explorer)

치아에 이상이 있는지 알아보거나 접근이 어려운 부위를 감지

핀셋 (pincette)

구강에 작은 재료를 넣거나 제거 (커튼 플라이어, cotton plier, forceps)

② 유닛 체어 부속 장비
장비기능·특징
진공 흡입기 (suction)물·침·구강 조직·이물질 흡입 / 1회용 석션 팁(고무·플라스틱): 내부 와이어로 구부려 사용 가능 / 금속 석션 팁: 멸균하여 사용 / 환자마다 교체
핸드피스 (handpiece)고속 또는 저속으로 버(bur)를 회전시켜 치아 삭제
공기물 사출기 (3 Way syringe)좌측 = 물 / 우측 = 공기 / 동시에 누르면 = 분사 형태 물
스케일러 (scaler)초음파(소리를 진동으로 변화)로 치아 표면 치석 제거
조명 (light)시술 부위 비춤 / 환자 눈에 직접 비춰지지 않게 / 환자와의 거리 60~90cm 유지
타구 (spittoon)시술 중 물·침 뱉는 곳, 자동 세척 / 청결 관리 철저 필요
기구 선반 (bracket table)진료 기구·재료·약품 올려놓는 곳, 위치·높낮이 조절 가능
발 조절기 (foot controller)핸드피스 회전 속도 조절, 진료 의자 높이·등받이 조절
③ 기타 기구
기구 목록
  • 엑스카베이터(excavator) — 숟가락 형태, 치아우식 진행된 치질(상아질) 제거
  • 트레이(tray) — 금속·플라스틱, 구강 모형 제작(인상 채득) 시 사용
  • 치주 탐침(periodontal probe) — 치주낭 깊이 측정, 억지로 넣지 말 것
  • 구강 모형(model) — 석고로 대상자 구강 모양 재현
  • 교합기(articulator) — 악관절 운동 재현, 구강 모형 결합
  • 교합인기(bite) — 왁스·레진 꽉 물게 → 교합기에 모형 결합
  • 발치 겸자(extraction forceps) — 발치 시 치아 잡는 기구
  • 발치 기자(extraction elevator) — 지렛대 원리로 치주인대로부터 분리
간호조무사의 업무
  • 진료 전: 대상자 안내, 진료실 관리, 기구 소독 및 준비
  • 진료 중: 진료 보조, 진공 흡입기 사용, 기구 및 재료 전달
  • 진료 후: 추후 진료 예약, 주의 사항 전달·교육, 기구·재료 정리
진공 흡입기 사용
  • 치아 순측 삭제 시 → 진공 흡입기는 설측에 위치
  • 의사가 오른손으로 핸드피스 잡으면 → 간호조무사도 오른손으로 잡음
위치 선정
  • 시술자 영역: 7~12시
  • 기구 교환 영역: 4~7시
  • 보조자 영역: 2~5시
  • 진료 시 시술자 팔꿈치 = 환자 구강 높이
④ 기구 전달 방법
  • 진료 순서 이해하여 미리 기구 준비
  • 시술자가 기구를 받아서 위치를 바꾸지 않도록 부위에 맞게 전달
  • 엄지·검지·중지로 기구를 건네고 약지·애지로 사용 중이던 기구를 받음
  • 이동기구함은 손이 닿는 거리에 둠 / 석션 팁은 환자 바뀔 때마다 교체 / 진료 중 진공 흡입기 적절히 사용
📌 Page 5 — 치료 종류·감염관리·단원정리
① 방습법
러버댐 방습법 (rubber dam)
  • 얇은 고무판에 구멍 → 해당 치아만 노출, 나머지 격리
  • 시야 확보, 시술 부위 건조·깨끗하게 확보
간이 방습법
  • 솜 또는 거즈를 상악에는 치열과 협벽 사이, 하악에는 혀 아래에 삽입
  • 타액 흡수 목적
② 보철 치료
종류설명
인레이 (inlay)일부 손상된 치관 부위에 와동(구멍) 만들어 아말감·금·도재·레진 끼워 넣음
크라운 (crown)손상된 치관 전체 삭제 후 금/도재 등으로 치관 전체 씌움 (하나의 치아)
브리지 (bridge)크라운과 동일하나 하나 이상의 치아
부분 틀니 (PD/RPD)치열 중 일부 치아 상실 시 (가철성 국소 의치)
전체 틀니 (FD)전체 치아 상실 시 (총의치)
임플란트 (implant)치아 전체(치관+치근) 인공 교환 / 턱뼈에 티타늄 fixture 식립 → Abutment → Crown
틀니 보관: 찬물이나 미온수 담긴 뚜껑·이름표 있는 통에 보관 (변형 예방)
틀니 세척: 싱크대에 수건 깔고 전용 세제 사용 (치약 금지 — 연마제로 마모 발생, 뜨거운 물 금지 — 변형)
③ 발치 후 주의 사항 & 기타 치료
발치 후 주의 사항 (핵심 10가지)
  • 거즈 약 2시간 물고 있기
  • 외과수술 후 48시간 냉찜질 → 부종·통증 감소
  • 뜨거운 목욕·무리한 운동 금지
  • 빨대 사용·흡연·침 뱉기·입 안에서 모으는 동작 금지
  • 침·혈액은 입 벌려 부드럽게 흘러내리도록
  • 발치 부위 칫솔질 대신 생리식염수·양치액으로 가볍게 헹구기
  • 처방된 약 지시대로 복용
  • 약간의 출혈·통증은 마취 풀리면서 발생 가능
  • 심한 통증·출혈 지속 시 치과 방문
  • 정해진 날짜에 치과 방문 → 소독·봉합사 제거
근관 치료·미백·방사선 촬영
  • 근관 치료(신경 치료): 우식·외상으로 치수 손상 시 치수 제거 → 충전 → 보철 치료
  • 미백(whitening): 전문가 미백 (치과) + 가정용 미백
  • 치근단 촬영: 일부 치아 및 주위 조직 관찰 (구강 내 필름 위치)
  • 파노라마 촬영: 전체적인 상태 관찰 (턱 올려놓고 촬영)
  • CT: 위치·깊이를 X-ray보다 정확히 알 수 있고 3차원적 관찰 가능
④ 감염 관리
멸균법 4가지
방법적용 기구
고압증기멸균 (autoclave)대부분의 외과용 기구, 날 없는 금속 기구 / 치과 가장 많이 이용 / 치경·유리제품도 해당
건열멸균 (dry heat)교정용 기구류, 유리 제품
화학멸균 (chemical)버(bur), 근관용 기구류
불꽃 소독알코올 램프 불꽃에 직접 소독 (치과에서만 사용)
개인 위생·안전 수칙
  • 손 위생: 환자 진료 전·후, 혈액·타액·분비물 오염 시, 진료구역 떠나기 전
  • 복장: 반지·팔찌·시계 착용 금지, 손톱 청결·짧게, 머리 묶거나 핀 고정
  • 보호 장비: 앞치마, 고무 장갑, 마스크, 보안경, 안면 보호대, 가운
  • 예방 접종: 인플루엔자, B형 간염
  • 바늘 뚜껑 벗겨진 주사기 주고받기 금지
  • 바늘 끝이 몸 쪽으로 향하지 않도록
  • 바늘 뚜껑 닫을 때 한손 뜨기 기법
  • 날카로운 재료는 전용 용기에 버림

✦ 핵심 암기
  • 타액선 크기: 이하선 > 악하선 > 설하선
  • 인체에서 가장 단단한 부분 = 사기질(법랑질)
  • 신경·혈관이 있는 치아 조직 = 치수(pulp)
  • 유치 총 20개 / 영구치 총 32개
  • 치아 교환 시작 = 하악 유중절치부터
  • 제1대구치 = 6세구치 / 제2대구치 = 12세구치 / 제3대구치 = 사랑니
  • 상수도 불소화: 1ppm → 우식 60% 예방
  • 치과 가장 많이 이용하는 멸균 = 고압증기멸균(autoclave)
  • 불꽃 소독은 치과에서만 사용
  • 보조자 의자 높이 = 시술자 의자보다 약 10cm 높게
