📌 Page 1 — 영양·안전·운동·수면
① 노인의 영양 권장량
탄수화물
  • 총 에너지의 55~65%
  • 단당류보다 복합 탄수화물 권장 (과일, 야채, 현미)
단백질
  • 총 에너지의 7~20%
  • 소화율은 감퇴, 흡수율은 청장년과 큰 차이 없음
  • 과다 섭취 → 칼슘 배설 증가 → 골격 손실
지방
  • 에너지 섭취량의 20%
  • 과잉 → 동맥경화·심혈관 질환 발생 증가
  • 비타민·무기질: 노년기에도 큰 변화 없음
  • 폐경기 이후 여성·남성 노인의 철분 요구량 같음
② 영양 문제와 간호 보조
문제원인간호 보조
연하 곤란 식도 운동 변화, 뇌졸중, 침 분비 감소, 의치, 불안 묽은 액체보다 농도가 진한 음식 제공 / 소량씩 자주 / 식사 전후 15~30분간 앉기 / 편마비 시 마비 없는 쪽으로 음식 넣어줌 / 식사 중 말하지 않도록
체액 불균형 (탈수) 콩팥 기능 저하, 갈증 덜 느낌, 요실금 두려움 하루 6~8잔 물 마시기 / 차·주스·우유 등 음료 / 수프·아이스크림·과일 등 수분 많은 음식
비만 65세 이상: 표준체중 20% 초과 또는 BMI ≥ 29 식이 제한 + 적당한 운동 병행
영양 결핍 질환, 투약, 알코올, 구강 건강 문제, 식욕 부진, 재정 문제 전문가 의뢰, 사회화 증진, 영양 교육, 구강 건조 예방
변비 섬유소·수분 부족, 운동 부족 하루 1,500~3,000mL 수분 / 아침 식전 물·과일 주스 / 섬유질 풍부 식품 / TV 시청 줄이고 운동 / 규칙적 배변 습관
골다공증 칼슘 섭취 부족, 흡수율 감소, 칼슘 배설 증가 / 여성: 노화·에스트로겐 부족 양질의 단백질 식품 / 폐경기 후 여성 칼슘 권장량 800mg/일 / 체중 부하 운동
심리적 영양 불량증 (FTT) 살아야겠다는 의지 감소 + 영양 장애 생리적·사회심리적·영적인 면 분석 / 걱정 표현하도록 경청 / 삶에 대한 긍정적 평가 / 자가 간호 동기 유발 / 사회 단체 서비스 연결
③ 낙상 예방·운동
낙상 원인
  • 노화: 시력 감소, 백내장, 보행·조절 기능 장애, 잦은 화장실 출입
  • 부적절한 기동성 보장구 사용
  • 약물: 항고혈압제·진정제·이뇨제 등 → 현훈·체위성 저혈압·실금
  • 잘 맞지 않는 신발·헐거운 의복
  • 체위성 저혈압, 허약, 뼈 약화, 운동 실조증
  • 젖은 바닥, 어지럽게 널린 물건, 희미한 조명
운동의 종류
  • 유산소 운동: 걷기, 자전거 타기, 평영으로 수영 / 걷기 = 심장 부하가 가장 적은 운동
  • 근력 강화 운동: 골밀도 높이고 근골격계 기능 개선 / 팔굽혀 펴기, 선 자세로 다리굽혀 펴기, 고정식 자전거 타기, 수중 걷기
  • 유연성 운동: 스트레칭 → 유연성 향상, 통증 예방·치료
  • 균형 운동: 낙상 예방 / 한 발로 서기, 눈 감고 서기, 태극권, 균형대 운동
  • 운동 강도: 개인 최대 능력의 40~85% / 매주 3~5회 / 매회 30~40분
④ 노인 수면 관리
노인 수면 특징
  • 일찍 자고 일찍 일어남
  • 수면량 줄어들고, 총 침상 시간은 길어짐
  • 수면 중 자주 깨어남, 잠 들기까지 시간 오래 걸림
  • 노화에 따라 REM 수면 감소
  • NREM 수면: 1단계 증가, 3·4단계 감소
수면 관리
  • 아침 기상 시간을 일정하게
  • 카페인 음료(커피·녹차·홍차) 오후 금지
  • 금주, 금연
  • 저녁의 과식은 숙면 방해
  • 공복감 시 따뜻한 우유
  • 수면제·진정제 장기 복용 금지
  • 매일 규칙적·적절한 운동
  • 낮잠을 많이 자면 밤잠 설침 → 낮잠 삼가기
📌 Page 2 — 약물 관리·노인 학대·우울증·의사소통
① 약물 관리
  • 노인 인구 대부분은 만성 질환 1가지 이상 → 복용 약 종류 많음 → 약물 상호 작용·부작용 위험 증가
  • 노화에 따른 신장 기능 저하, 간 기능 감소 → 약의 흡수·분포·대사·배설 신중히 고려
  • 동일 약물이라도 청장년과 다른 약리 작용·부작용 나타날 수 있음
  • 노인의 기억력 감소 → 약물 복용 시간·방법·보관 등 오용 위험
노인의 약물 이행을 높이는 방법
  • 약물 복용 횟수를 하루 1회 또는 2회로 처방
  • 복용 시간을 식사 후 또는 잠자기 전 등 일상생활에 맞춤
  • 보호자에게 약물 이름·부작용 설명
  • 약을 담는 용기, 설명서, 약 복용 달력 제공
  • 약물 복용의 필요성을 지속적으로 교육
  • 의사 처방 없이 살 수 있는 일반 의약품 복용 가능성 고려
  • 치매 및 우울증 등 인지 기능 문제에 대해 정기적으로 평가
② 노인 학대
학대 유형 (7가지)
유형예시
신체적 학대때림, 꼬집기, 물건 집어던지기, 신체적 구속·감금, 화상 입힘
정서적 학대욕설·고함, 무시, 비웃거나 조롱, 외출시키지 않기
성적 학대성 관련 언어 표현·행위, 강제적 성행위
경제적 학대유언장 허위 작성, 허락 없이 재산 매매·계좌 해약
방임필요한 의료 처치 받게 하지 않음, 생계비 지원 안 함
자기 방임노인 스스로 치료 행위 거부, 의·식·주 관련 행위 거부
유기낯선 장소에 버림, 시설에 맡기고 연락 안 함
학대 예방·신고
  • 노인복지법 — 누구든지 노인 학대를 알게 된 때 신고 의무
  • 신고 기관: 노인 보호 전문 기관 또는 수사 기관
  • 미신고 시: 300만 원 이하 과태료
  • 세계 노인 학대 인식의 날: 매년 6월 15일
  • 한국: KINPEA 조직·활동
예방을 위한 접근
  • 노인 개인: 의존성 줄이고 자립심 증대
  • 가족: 돌봄 부담 경감 지원, 노인 특성 이해
  • 사회: 법적·제도적·사회복지 서비스 연계
③ 우울증 관리
우울증 원인·특성
  • 노인 자살의 주 원인
  • 남성에 비해 여성이 많음
  • 소득이 낮은 사람들, 독거노인: 우울증 위험 높음
  • 원인: 신체 기능 저하에 따른 허무감, 역할 감소, 빈곤, 배우자 사망, 고독감
치료 및 간호
  • 평소 건강검진 때 우울증 사정
  • 즉각적인 치료로 회복 빠를 수 있음
  • 정신치료·항우울제: 다양한 정도의 우울증 완화
  • 전기 경련 요법: 다른 치료에 반응하지 않는 심각한 우울증에 효과적
  • 적절한 영양·규칙적인 운동
  • 독거·사회적 고립·열악한 주거·경제적 빈곤·학대·방임 등에 적극 개입
  • 사회적 지지 자원·프로그램 참여 연계
④ 효과적인 의사소통
언어적 의사소통
  • 기술: 경청, 질문, 반영, 공감, 설명, 자기 주장, 유머
  • 노인의 관심에 집중, 생각·감정 충분히 듣고 느끼기
  • "아, 예." "그렇군요." 지지적 언어 반응 / 고개 끄덕이기
  • 질문 후 생각할 수 있는 시간을 주고 느긋하게 반응 기다림
  • 상담 시 질문: 구체적·간결하게, 개방적 질문 선택
비언어적 의사소통
  • 인지·감각 기능 저하 노인의 비언어적 메시지는 언어적 메시지보다 더 많은 정보 제공
  • 적절한 눈 맞춤: 자기 표현 원활 / 과도한 눈 맞춤은 불편감
  • 노인 이야기는 대개 반복적·길어지는 경향 → 분산된 초점 요약하고 일관된 방향으로 유도
📌 Page 3 — 골절·대사증후군·치매
① 골절 (fracture)
  • 외부의 힘에 의해 뼈의 연속성이 완전·불완전하게 소실된 상태
  • 노인 골절: 낮은 강도의 외상으로 발생, 주로 실내에서 발생
  • 취약한 이유: 골다공증으로 뼈 약화, 균형 감각 저하, 낙상 시 보호 반사 기능 두드러지게 약화
  • 여성 노인: 골밀도 낮아 주로 대퇴골 근위부 골절 많이 발생
  • 고관절 골절 장기 부동 → 폐렴, 뇌졸중, 욕창, 영양 실조 → 사망률 크게 증가
간호 보조
  • 주기적으로 골절 부위 피부 상태 살펴 욕창 조기 파악·치료
  • 조명 밝게, 걸려 넘어질 물건 치우기, 욕실 손잡이 설치, 매트 깔기, 계단 이동 시 난간 잡기
  • 자전거 타기, 걷기, 스트레칭 등 규칙적 운동
  • 석고 붕대: 사지 24~48시간 상승 → 부종 예방 / 손가락·발가락 자주 움직여 정맥 귀환 촉진
② 대사 증후군 (metabolic syndrome)

복부 비만, 인슐린 저항성으로 인한 혈당 상승, 고혈압, 고지혈증 등 심혈관 질환의 주요 위험인자를 동시에 가진 질환군

대사 증후군 진단 기준 (5개 항목 중 복부 비만 필수 + 2개 이상)
항목기준
복부 비만 (필수)남성: 허리 둘레 36인치(90cm) 이상 / 여성: 34인치(85cm) 이상
고중성 지방 혈증중성 지방 150mg/dL 이상 또는 치료제 복용
낮은 HDL-콜레스테롤남성: 40mg/dL 미만 / 여성: 50mg/dL 미만 또는 치료제 복용
고혈압수축기/이완기 혈압 130/80mmHg 이상 또는 고혈압 약물 복용
고혈당공복 시 혈당 100mg/dL 이상 또는 당뇨 약물 복용
치료 및 간호 보조
  • 지방: 총 칼로리의 25% / 포화지방·트랜스지방 줄이기 / 불포화지방 적당히 유지
  • 콜레스테롤 200mg 이하
  • 혈압 조절 위해 염분 제한
  • 혈당 조절 위해 단당류 대신 복합 탄수화물(다당류)
  • 운동 및 신체 활동 증가 (심혈관 질환 있으면 운동 전 검사)
  • 하루 2잔 이상 음주 금지 / 반드시 금연
③ 치매 (dementia)
정의·원인·증상
  • 여러 가지 원인의 뇌 손상에 의해 인지 기능을 상실하여 일상생활 수행 불능
  • 의식은 흐려지지 않는 것이 특징
  • 알츠하이머 병: 원인 미상의 신경 퇴행성 질환 → 50~60%
  • 혈관성 치매: 뇌 혈액 순환 장애 → 20~30%
알츠하이머 vs 혈관성 치매
알츠하이머혈관성 치매
발병·악화점진적급작스러운 발병, 계단식 악화
초기 증상기억력 장애가 두드러짐인지 기능의 심한 장애 (실행 기능, 언어 장애)
신경학적 이상병이 상당히 진행한 후초기부터 동반 (걸음걸이 불안정, 요실금)
치매 환자 돌봄 · 의사소통
  • 대상자의 안전을 위협하는 요인 확인·제거
  • 안전하고 익숙하며 단순한 환경 조성
  • 일과표 만들어 일관되고 반복적인 생활
  • 일상생활에서 잔존 기능 최대한 활용
  • 칭찬·격려, 자존심 건드리지 않기
  • 천천히 치매 환자 속도에 맞추기
  • 쉬운 단어, 짧은 문장, 명확하게
  • 틀렸다고 지적하지 않음 / 한번에 하나씩
  • 대답 충분히 기다리기
📌 Page 4 — 욕창·망막질환·장기요양 서비스
① 욕창 (pressure sore)

오랫동안 뼈 돌출 부위와 침대·의자 간에 압박을 받아 생긴 국소적인 연조직 괴사

원인
부동 영양 불량 습기 마찰력 응전력 지각 감각 손상
단계특성욕창 관리
1단계 상피층에 국한된 급성 염증, 불규칙한 홍반·표재성 궤양, 주로 뼈 튀어나온 부위 압력·습기 피하기 / 국소 피부 관리 집중
2단계 진피층까지 확산, 수포·박리 또는 얕은 궤양, 초기 섬유화·변색 상처가 심해지지 않게 / 국소 피부 관리
3단계 피하 지방까지 침범하는 피부 전층, 피부가 아래로 잠식, 괴사 조직·악취 괴사 조직 제거 / 상처 깨끗이 보호 / 필요 시 세균 배양 검사·국소 항생제
4단계 근막을 지나 확대되어 뼈가 보임, 골수염·패혈성 관절염 가능 조직 생검 / 봉와직염·골수염에 전신 항생제 / 수술적 치료 고려
욕창 예방 관리 핵심
  • 뼈 돌출 부위가 바닥에 직접 닿지 않게
  • 발뒤꿈치 베개나 쿠션으로 올리기
  • 침대 머리 부분 최소한으로 올리기 (응전력 감소)
  • 1~2시간마다 체위 변경
  • 도넛형 쿠션 사용 금지 (정맥충혈·부종 유발 → 욕창 위험)
  • 뜨거운 물·과도한 피부 마찰 금지
  • 뼈 돌출 부위 마사지 금지
  • 요실금 시 즉시 세척, 수분 차단 크림, 흡수 패드 사용
② 망막 질환 3가지
노인성 황반 변성
  • 황반 기능 저하 → 시력 저하·상실
  • 여성에게 더 많이 발생
  • 65세 이상 시력 소실의 가장 흔한 원인
  • 종류: 건성(90%, 서서히 진행) / 습성(빠르고 심각)
  • 증상: 변색증(물체 색 이상), 변시증(굽어 보임), 시야 가운데 흐림
  • 고농도 황산화제·아연 음식 → 진행 늦출 수 있음 (회복 안 됨)
당뇨성 망막 병증
  • 당뇨로 망막 미세 혈관 손상
  • 비정상적인 망막 혈관의 증식
  • 증상: 초기 증상 없음 → 중심 시력 손상 진행 → 망막 박리 → 실명
  • 조기 발견이 중요
  • 당뇨 진단 환자: 매년 안과·내과 검진
  • 치료: 혈당·혈압 조절, 금연, 레이저 광응고술
망막 박리
  • 노화로 유리체가 망막으로부터 밀려나고 위축 → 망막 찢어짐 → 분리
  • 원인: 노화, 외상, 고도근시, 당뇨병, 고혈압
  • 증상: 통증 없음 / 광시증(빛 번쩍) / 비문증·날파리증 / 시야 그림자·커튼
  • 치료: 약물, 레이저, 수술 (대부분 수술 필수)
  • 수술: 냉응고술, 레이저 광응고술, 망막 수술
③ 장기 요양 서비스

고령이나 노인성 질병(치매·뇌혈관 질환·파킨슨병 등)으로 6개월 이상 다른 사람의 도움 없이 일상생활이 어려운 대상자에게 제공

  • 시설 급여 (본인 부담금 20%): 노인 요양 시설, 노인 요양 공동 생활 가정
  • 재가 급여 (본인 부담금 15%): 방문 요양, 방문 간호, 방문 목욕, 주·야간 보호, 단기 보호
  • 가족 요양비: 매월 15만 원 지급
  • 이용 대상: 장기 요양이 필요한 만 65세 이상 또는 65세 미만 노인성 질병 보유자
  • 등급 판정: 1~5등급
  • 신청: 국민건강보험공단 지역별 지사
  • 문의: ☏ 1577-1000 / 치매 상담 ☏ 1899-9988

① 퀴블러 로스의 죽음 5단계
단계특징간호 중재
1. 부정 자신의 죽음이 임박했다는 사실 부인 / 오진이라고 생각 / 사실을 의논하기 원하면서도 현실 부정 반복 개인의 반응 수용 / 열린 마음으로 정직하게 / 개방된 분위기 제공 / 정보 내면화·방어 시간 허용
2. 분노 주위 사람이 분노의 대상이 됨 / "모두 옳지 않다"는 표현 환자·가족 불편감 완화 / 환자 행동 이해 돕기 / 분노에 화내거나 모욕적으로 반응하지 않기 / 감정 표출을 당연한 과정으로 수용
3. 타협 생의 연장과 관련, 비밀리에 신과 이루어지거나 기도를 통해 표현 타협이 무시될 경우 실망·죄의식 느낄 수 있음을 간과하지 않기
4. 우울 기능 상실·증상 악화 경험 / 노인은 여러 상실 경험으로 우울에 잘 빠짐 손을 잡고 말없이 앉아 있기 (활발한 대화보다 의미 있음) / 조용하게 명상하도록 함께 / 환자가 우울 극복하여 수용·평온 상태에서 죽음에 다가갈 수 있도록
5. 수용 안정을 찾고 죽음이라는 용어 사용 / 평화로워 보임 (행복한 상태 아님) 비언어적 의사 소통이 유용 (단순한 접촉, 위로, 함께 있기) / 희망은 임종 과정 모든 단계에 나타날 수 있음
생애 말기: 현대 의학으로 통상의 치료를 해도 치유되지 않고 6개월 이내에 죽음이 다가올 것으로 예상되는 상태
사전 연명 의료 의향서(ACD): 환자가 미리 작성한 서면 진술서 (장점: 환자 권리 보호, 가족 결정 부담 감소, 만족도 향상)

③ 빈출 암기 포인트 🔥
✦ 핵심 암기
  • 노인 탄수화물 권장: 총 에너지의 55~65% / 지방: 20% / 단백질: 7~20%
  • 연하 곤란: 묽은 것보다 농도가 진한 음식 / 식사 전후 15~30분 앉기
  • 노인 변비 수분: 하루 1,500~3,000mL
  • 골다공증 폐경 후 여성 칼슘: 800mg/일
  • 노인 비만: 표준체중 20% 초과 또는 BMI ≥ 29
  • 욕창 체위 변경: 1~2시간마다 / 도넛형 쿠션 금지
  • 욕창 석고 붕대 사지 상승: 24~48시간
  • 노인 학대 미신고 과태료: 300만 원 이하
  • 세계 노인 학대 인식의 날: 매년 6월 15일
  • 대사 증후군: 복부 비만(필수) + 나머지 4개 항목 중 2개 이상
  • 알츠하이머 초기 = 기억력 장애 / 혈관성 치매 초기 = 인지 기능 장애 + 신경학적 이상
  • 퀴블러 로스 5단계: 부정→분노→타협→우울→수용
  • 생애 말기 = 6개월 이내 사망 예상
  • 장기 요양 시설 급여 본인 부담금: 20% / 재가 급여: 15%
④ 자주 틀리는 선지 ⚠
틀리기 쉬운 것들
  • 도넛형 쿠션 욕창 예방에 효과적 → ✗ (정맥충혈·부종 유발 → 욕창 위험 커짐)
  • 욕창 시 뼈 돌출 부위 마사지 → ✗ (마사지 금지)
  • 노인 연하 곤란: 묽은 액체 음식 제공 → ✗ (농도가 진한 음식 제공)
  • 노인은 총 수면 시간이 늘어남 → ✗ (수면량은 줄어들고 총 침상 시간은 길어짐)
  • 퀴블러 로스 타협 단계 = 우울한 상태 → ✗ (생의 연장을 기원하는 단계)
  • 수용 = 행복한 상태 → ✗ (평화로워 보이지만 행복한 상태를 의미하지 않음)
  • 눈 감고 서기 = 근력 강화 운동 → ✗ (균형 운동에 해당)
  • 알츠하이머 초기에 신경학적 이상 동반 → ✗ (혈관성 치매가 초기부터 동반)
