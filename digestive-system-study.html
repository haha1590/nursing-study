📌 Page 2 — 식도·위 질환
① 역류성 식도염 (Reflux Esophagitis)
정의 & 병태생리
  • 위산을 포함한 위 내용물이 식도 쪽으로 역류하여 식도 점막을 자극하는 질환
  • 식도와 위 사이 분문(식도 괄약근)의 조임 힘이 약해져 발생
  • 높은 위산이 식도 점막을 지속적으로 자극
증상
  • 가슴앓이 — 흉골 하부~턱 부위까지 타는 느낌
  • 소화불량, 기침, 호흡곤란
  • 쉰 목소리, 질식감, 가슴 통증
진단
  • 식도조영 검사, 상부 위장관 내시경
  • 24시간 식도 산도(pH) 감시법
  • 정상 pH 6.5~7.0 → 산 역류 있으면 pH 4.0 이하로 감소
  • 생검·세포학적 검사 (바렛식도·위암·식도암 감별)
간호 보조
  • 침상 머리 30° 상승 (중력으로 위 자연스럽게 비워지게)
  • 식사 후 2~3시간 동안 눕지 않기
  • 탄산음료, 빼대로 음료수 마시기, 가스발생 음식 제한
  • 식후 몸을 앞으로 구부리거나 무거운 물건 들기 금지

식이 저지방, 고섬유식 / 식사 시 물과 함께 고단백, 고지방, 저탄수화물 / 식전 1~2시간 수분 섭취 제한
② 위염 (Gastritis)
정의 & 병태생리
  • 위 점막의 염증
  • 정상 위 점막 장벽에 이상 → 염산과 펩신이 위 조직 자극
  • 원인: 과식, 스트레스, 알코올, 지나친 카페인
증상 & 진단
  • 식욕부진, 오심과 구토, 상복부 압통
  • 복부 팽만감
  • 위내시경으로 위염 분류
  • 원인 감별 필요 시 조직 검사
간호 보조
  • 약물 치료 (위 점막 손상 감소)
  • 비위관 간호 → 출혈 관찰 or 위 내부 세척
  • 맑은 유동식 → 부드러운 음식 → 고형식 점진적으로 섭취
  • 지극적 음식, 카페인, 과식 금지
  • 위염은 위암으로 발전 가능 (4단계 자연 발생)
③ 소화성 궤양 (Peptic Ulcer) — 위 점막 + 근육층까지 손상
정의 & 병태생리
  • 위장 점막이 손상되어 나타나는 질환
  • 위산·펩신 분비 vs 점막 방어기전 불균형으로 발생
  • 헬리코박터 파일로리균 감염 / NSAIDs 복용으로 점막 방어 손상
증상
  • 궤양 부위에 위산이 닿으면 평활근 수축 → 통증
  • 둔하거나 찌르는 듯한 느낌, 쓰린 통증
  • 구토 — 유문 근육 경련 및 기계적 폐쇄로 발생

진단
  • 상부 위장관 내시경 + 생검
  • 요소호기 검사 — H. pylori 감염 여부 확인 (요소 → 암모니아 + 이산화탄소로 변환 능력 이용)
간호 보조
  • 항궤양성 약물 급하게 중단 금지 (위산 심하게 분비될 위험)
  • 아스피린·NSAIDs 위궤양·위궤양 발생 → 피할 것
  • 커피·차·콜라 → 통증 유발 / 알코올 → 위산 분비 촉진 / 흡연 → 궤양 재발률 증가
  • 소량씩 자주 일정한 간격으로 섭취
⚠ 덤핑 증후군 (Dumping Syndrome)
  • 위 대부분 절제 또는 유문괄약근 제거 수술 직접 결과로 나타남
  • 위 내용물이 소장으로 빨리 내려감 → 고삼투압 음식으로 소장 내 수분 이동 → 혈장량 감소 → 장 팽만
  • 증상: 식사 후 15~30분 후 발생 / 허약, 발한, 심계항진, 현기증
  • 치료: 앙와위, 고단백·고지방·저탄수화물, 식전 1~2시간 수분 섭취 제한
📌 Page 3 — 충수돌기염·간 질환
① 충수돌기염 (Appendicitis)
정의 & 병태생리
  • 맹장 하부에 연결된 충수에 염증 발생
  • 축적된 변에 의한 충수관 폐색으로 나타남
증상
  • 식욕부진, 오심, 구토 뒤따르는 통증으로 배꼽 주위에서 시작
  • 지속적이며 우하복부(RLQ)로 이동
  • McBurney's point — 배꼽과 오른쪽 장골능 잇는 선상의 중간점
진단 징후
징후설명
반동 압통 압박 후 빨리 손 뗄 때 통증
McBurney 징후 배꼽에서 우측 전상장골극에 압통·강직
Rovsing 징후 McBurney's point 반대(좌측) 압박 시 McBurney's point 통증
요근 징후 앙와위 뒤로 과신전 시 우하 사분면 통증
폐쇄근 징후 앙와위 굴곡 후 안쪽으로 회전 시 통증
진단: 신체검진, 초음파, CT 스캔
치료: 하제나 관장은 연동 운동 증가 → 충수 파열 위험 → 금지!
② 간염 (Hepatitis)
정의 & 병태생리

간염 바이러스에 의한 급격한 간세포 장애


  • 세포독성 시토카인과 자연살해세포(NK cell)가 감염된 간세포 용해 → 간 손상
  • 항원-항체 복합체가 전신 순환 → 보체 활성화 → 발진, 혈관부종, 관절염, 고열, 권태감
용어 설명
  • 시토카인: 세포·주로 백혈구에서 분비되는 단백활성 물질
  • 자연살해세포: 바이러스에 감염된 세포나 암세포를 직접 파괴하는 면역세포
  • 보체: 산선한 혈액·림프액 속에 함유된 효소 모양의 단백질의 일종

증상
  • 한 달~4개월 잠복기 동안: 권태감, 식욕부진, 체중감소
  • 피로, 오심, 간헐적 구토, 복부 불편감
  • 발열, 황달, 소양증, 간수치 상승
치료 및 간호
  • 절대안정, 수분 섭취 증가
  • 고탄수화물·고단백·고비타민·저지방·저염 식이
  • 저지방·고탄수화물·적정 단백식이를 소량씩 자주 제공
  • 추가적인 간 손상 방지 → 휴식
③ 간경변증 (간경화증, Liver Cirrhosis)
정의 & 병태생리
  • 만성적인 염증으로 정상 간 조직이 섬유화되어 간 기능 저하
  • 간염 → 간경변 → 간암 진행 가능
증상
  • 간염 증상, 비장 비대
  • 복수, 핍뇨
  • 위장·식도 출혈, 저알부민혈증
  • 하지 부종, 심하면 간성혼수
치료 및 간호
  • 식이: 고단백 (간성혼수 시 단백질 제한), 고탄수화물, 저지방, 저염
  • 복수·부종 시 수분과 염분 제한
  • 다량의 비타민B 복합제 투여
  • 소량씩 자주 섭취
  • 매일 체중·수분 섭취량·배설량 측정
  • 소양증 완화 피부 간호
  • 금연, 금주, 침상 안정, 출혈 예방
  • 복수 시 복수 천자
  • 항히스타민제, 이뇨제 처방 약물 투여
⚠ 간성뇌증 (간성혼수)
  • 간기능 장애로 의식 저하, 행동 변화
  • 원인: 단백질 분해 시 암모니아 과다 / 변비 / 위장관 출혈
  • 증상: 혼수, 착란증, 경련, 사지 떨림, 현기증, 호흡 시 단 냄새, 혈중 암모니아 증가
  • 반드시 저단백 식이
  • 락툴로오즈(듀파락) 복용
  • 같은 용액으로 정체관장 시행 → 암모니아 배출
④ 황달 종류
종류원인특징적 증상
폐쇄성 황달 담도 폐쇄 황달 + 소양증 + 회백색 대변
비폐쇄성 황달 간세포 손상 → 담즙 생산 저하 황달 (대변 색깔 크게 변하지 않음)
용혈성 황달 적혈구 파괴 황달 + 빈혈 증상
혈액 내 담즙색소(빌리루빈) 농도가 비정상적으로 증가 → 피부·점막이 노랗게 됨 / 피부에 담즙산염 쌓임 → 소양증 발생
📌 Page 4 — 담낭·췌장 질환
① 담낭염 (Cholecystitis)
정의 & 병태생리
  • 담석이나 침전물에 의한 폐쇄로 나타남
  • 급성기: 담낭이 붓고 충혈 → 담즙이나 농으로 팽창
  • 담낭관이 침범되어 폐쇄될 수 있음
증상
임상 증상원인
폐쇄성 황달 십이지장으로 담즙이 흐르지 않아 발생
소양증 피부조직에 담즙산염이 침착
출혈 경향 비타민K 흡수 감소 → 프로트롬빈 생성 감소
지방변 담즙 흐름 폐쇄 → 지방이 십이지장으로 분비 안 됨 → 지방 소화 안 됨
오른쪽 어깨·견갑골로 방사되는 우상복부 통증, 발열, 황달
진단
  • 복부 초음파, CT
  • 담관 조영술
  • ERCP (내시경적 역행성 담췌관 조영술): 담도·췌장 질환 진단·치료에 이용
치료 & 간호
  • 체외 충격파 쇄석술 — 체외에서 높은 에너지 충격파로 결석 분쇄
  • 복부 담낭절제술
  • 통증기전 설명, 통증 없을 때 휴식
  • 구토·팽만 경감 → 비위관 삽입 → 수분·전해질 균형 유지
📋 담석증 (Cholelithiasis) 추가
  • 콜레스테롤, 빌리루빈, 칼슘 등으로 구성된 담석이 담낭·담관 내에서 형성
  • 담석 자체만으론 해롭지 않지만 담관을 막으면 → 소화불량, 지방 대사 이상, 황달, 소양증
  • 치료: 저지방식이, 안정, 진통제 투여, 수술 (체외 충격파 쇄석술, 개복수술)
② 췌장염 (Pancreatitis)
정의 & 병태생리
  • 췌장의 급성 염증
  • 십이지장으로 배설되지 못한 췌장 소화효소가 췌장 안에서 활성화 → 췌장 스스로 소화되는 급성 염증성 질환
  • 원인: 만성 알코올 중독증, 담석증
증상
  • 지속적이고 쥐어짜는 듯한 상복부 통증
  • 가슴, 배꼽 사이 또는 배꼽 주위 → 옆구리·하복부로 방사통
  • 오심, 구토, 미열, 백혈구 증가, 저혈압, 빈맥, 황달
  • 터너 징후 (Grey Turner's sign) — 옆구리 푸르스름한 변색
  • 쿨렌 징후 (Cullen's sign) — 배꼽 주위 푸르스름한 변색
진단
  • 혈청 아밀라아제: 24~72시간 동안 상승 (다른 질환에서도 상승 가능)
  • → 급성 췌장염에서는 혈청 리파아제가 더 중요한 검사
  • 복부 초음파, X-선 검사법, CT 스캔
간호 보조
  • 복부 팽만 감소 → 비위관 흡인
  • 복부 팽만으로 인한 압력 완화 → 반좌위
  • 저단백, 저지방, 탄수화물 식이를 소량씩 자주
  • 고지방식·고열량식이 → 위액 분비 자극 → 제한
⚠ 췌장염 점상 출혈 징후 2가지 암기!
터너 징후 (Grey Turner's sign)
→ 옆구리의 푸르스름한 변색
순환하는 트립신에 의한 혈관 손상
쿨렌 징후 (Cullen's sign)
→ 배꼽 주위의 푸르스름한 변색
복막 뒤 출혈이 배꼽 쪽으로 전파
③ 과민성 대장 증후군 (IBS)
정의 & 진단 기준
  • 대장 내시경·X선 검사로 확인되는 특정 질환은 없지만 식사나 가벼운 스트레스 후 복통, 복부 팽만감 같은 불쾌한 소화기 증상이 반복되는 만성 질환
진단 기준

지난 3개월간 평균 주 1회 이상 반복적인 복통이 있고, 다음 3가지 중 2가지 이상의 증상이 나타날 때:

  • 복통이 배변과 연관성이 있다
  • 복통과 연관하여 배변 횟수가 변한다
  • 복통과 연관하여 대변의 형태(굳기)가 변한다
치료 방법
  • 대변 배출 규칙적으로 → 식이섬유 20g 함유 식이 매일 섭취
  • 변의 양을 증가시키는 약품 처방
  • 증상에 초점 → 대상자에게 증상·식이·스트레스 상황에 대한 일기 작성
  • 무엇이 과민성 대장증후군 증상을 유발하는지 구분할 수 있도록 도움
진단
  • 철저한 병력사정·신체검진 필수
  • 대변 검사, X-선, 대장경 검사 → 다른 질환 없음 확인
📌 Page 5 — 소화기계 암 & 단원 정리
① 소화기계 관련 암
위암 (Gastric Cancer)
  • 40세 이상 남성에서 발병률 높음
  • 위험 요인: H. pylori, 흡연, 가족력, 짠 음식, 훈제 식품, 과거 위 수술
  • 증상: 식욕 상실, 몸무게 감소, 구토, 팽만, 식사 후 지나친 배부름
  • 위 출혈 시 토혈, 흑색변, 빈혈
  • 진단: 내시경 생검 or 바륨 검사
  • 치료: 위 절제술 (가장 일반적) → 수술 후 연동 운동 돌아오면 소량 맑은 유동식 → 유동식 → 고형식 순으로 증가
대장암 (Colorectal Cancer)
  • 50세 이상에서 가장 흔함, 남녀 모두 발생
  • 위험 요인: 가족력, 노화, 흡연, 붉은 가공육 다 섭취, 식이섬유 섭취 부족, 운동 부족, 과음
  • 증상: 배변 습관·대변 굳기 변화, 대변에 점액·혈액, 흑색변, 복통, 빈혈, 몸무게 감소
  • 진단: 바륨 검사, CT, 내시경, 종양 표지자 검사
  • 치료: 종양 크기·전이 따라 수술 + 화학요법
간암 (Liver Cancer)
  • 원인: 만성 바이러스 간염, 간경화증, 독소 노출
  • 증상: 복통, 몸무게 감소, 구토, 황달, 배 안쪽 덩어리감
  • 진단: 초음파, CT 촬영, 생검
  • 치료: 종양 제거 수술, 화학요법, 방사선요법, 간이식
  • 예후: 암의 전이 여부에 달려 있음
췌장암 (Pancreatic Cancer)
  • 암 진단 후 5년 생존율이 가장 낮은 암
  • 원인: 흡연, 만성 췌장염, 당뇨병, 과다 음주, 고지방식이
  • 증상: 좌측 상복부 갉아먹는 통증 (낮보다 밤에 더 심함), 황달, 회백색 대변, 소변 변화, 당뇨 발생·악화
  • 진단: 혈액 리파아제·아밀라아제 수치, CEA·CA19-9 상승
  • 치료: 절제 수술 (암 크기·위치에 따라) / 수술 후 조기 이상·잦은 체위 변경
담낭암 (Gallbladder Cancer)
  • 남성보다 여성에서 4배 높게 나타남, 50세 이상 발생
  • 증상: 체중 감소, 식욕감퇴, 오심·구토, 허약, 우상복부 통증
  • 체액·전해질 균형 유지, 통증 완화 위해 진정제 투여 가능
  • 담낭절제술 시행 → 저지방식이 / 수술 4~6주 후부터 지방 섭취 가능
② 위장관 출혈 비교
구분상부 위장관 출혈하부 위장관 출혈
발생 부위 식도, 위, 십이지장 공장, 회장, 결장, 직장 출혈
원인 식도 정맥류, 식도염, 소화성 궤양, 출혈성 위염, 스트레스성 궤양, 위암 결장 직장암, 용종, 감염성 설사, 궤양성 대장염, 크론병, 항문 직장 누공
증상 토혈 혈변, 잠혈, 반유, 검사, 시, 대변이 선홍색에서 검은색으로 다량 출혈
간호 활력 증상 측정, 섭취량·배설량 측정, 영양 불균형 관찰, 소화·흡수 과정 변화 확인, 복부 팽만·경련·오심·구토 확인 대변 색깔 확인 (장의 출혈 위치 확인), 소변 검사, 환자의 불안 상태 사정, 정서적 지지 및 안정감 유지
③ 빈출 암기 포인트 🔥
✦ 자주 나오는 핵심 암기
  • 역류성 식도염 체위 = 좌위 / 덤핑 증후군 체위 = 앙와위
  • 충수돌기염 압통점 = McBurney's point (RLQ) — 하제·관장 금지!
  • 췌장염 점상 출혈 → 터너 징후(옆구리) + 쿨렌 징후(배꼽)
  • 췌장염 진단 = 혈청 리파아제가 아밀라아제보다 더 중요 (급성)
  • 담즙 = 소화효소 아님! → 지방 유화 작용
  • 간성뇌증 치료 → 저단백 식이 + 락툴로오즈(듀파락) + 정체관장
  • B형 간염: 항원(-) + 항체(-) → 예방 접종 필요
  • C형 간염 = 예방 접종 없음 / 75~85% 만성 C형으로 발전
  • 담낭암 = 여성에서 4배 높음 / 췌장암 = 5년 생존율 가장 낮음
  • 위내시경 전 금식 8~12시간 / 검사 후 구역질 반사 회복까지 금식
④ 자주 틀리는 선지 ⚠
틀리기 쉬운 것들
  • 기계적 소화 = 소화효소에 의해 화학적으로 분해 → ❌ (화학적 소화가 맞음)
  • 분절 운동은 소장·대장 모두에서 발생 → ❌ (소장에서만)
  • 복부 청진 순서 = 시진→타진→청진→촉진 → ❌ (시진→청진→타진→촉진)
  • 담즙은 소화효소이다 → ❌ (유화 작용만 함)
  • 충수돌기염 치료 시 하제나 관장 시행 → ❌ (파열 위험)
  • 간성혼수 시 고단백 식이 → ❌ (저단백 식이)
  • 역류성 식도염 환자 = 앙와위(누운 자세) → ❌ (좌위 / 머리 30° 상승)
단원 평가 해설 포인트
주요 단원 평가 문항 답

5번: 담낭염 → ① 지방변
6번: A형 간염 → ③ 오염된 음식 섭취
